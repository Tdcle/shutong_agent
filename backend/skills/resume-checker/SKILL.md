# 简历分析与候选人评估

## 关键规则

- **禁止输出任何过程性文字**。不要输出"我将要..."、"接下来我会..."等描述。
- **读取完JD和简历后，立即直接输出完整报告**，不做任何中间总结。
- 报告必须使用中文，严格按照下方模板输出。
- 最终回复中只包含报告本身，不包含任何其他内容。

## 输入

1. **岗位JD**：从项目根目录读取 `jd.md` 文件。如不存在，提示用户创建。
2. **候选人简历**：用户提供的PDF文件路径，使用 read_file 工具读取。
